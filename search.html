The search sidebar is the primary area for search-related actions, including creating a custom search, editing a custom search, deleting a custom search, selecting a search, and executing a search. It is displayed on the left edge of the screen on-hover, and remains displayed if the user decides to pin the sidebar.

It is comprised of two main areas - the search header and search content. The search header is where the user can either select a saved custom search or create a new search. The search content area is where the user can modify or execute the selected custom search.

The main sidebar has the following styling:

Width: 250px;
Height: 100%;
Background: #14223D;

The search header has the following styling:

Width: 100%;
Height: 160px;
Background: #156FB7;
Padding: 15px;

The search content has the following styling:

Width: 100%;
Height: calc(100% - 170px);
Background: transparent;
Padding: 15px;
